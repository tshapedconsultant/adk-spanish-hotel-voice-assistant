# Security policy

**Maintainer:** Andres Lage

If you discover a security vulnerability, please **do not** open a public issue with exploit details.

- Prefer [GitHub Security Advisories](https://docs.github.com/en/code-security/security-advisories) for this repository (if enabled), or
- Contact **Andres Lage** privately with a description, impact, and reproduction steps.

Include the affected component (e.g. Flask webhooks, Redis session store, Gemini integration) and version/commit when possible.
